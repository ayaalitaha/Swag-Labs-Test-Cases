 Overview

This repository contains a comprehensive set of **manual test cases** for the Swag Labs (SauceDemo) web application. The purpose of this project is to demonstrate real-world QA testing skills by covering key functionalities of an e-commerce system.

The test cases focus on validating the core workflows including login, product browsing, cart management, and checkout process.

 Objectives

* Validate functional requirements of the application
* Identify positive and negative test scenarios
* Ensure proper UI/UX behavior across pages
* Verify boundary and validation rules
* Cover end-to-end user journey from login to checkout


 Modules Covered

 1. Login Module

 2. Products Page

 3. Cart Module

 4. Checkout Module
